# 🧬 Whole-Exome Sequencing Variant Analysis Pipeline

Whole-exome sequencing pipeline for variant calling, filtering, and functional annotation of coding variants.

## 🎯 Project Goal
Developed a bioinformatics workflow to process exome sequencing data, identify coding variants, and annotate potentially functional mutations for downstream genomic interpretation.

## 📌 Overview
This project implements a standard WES analysis workflow including:
- sequence alignment
- BAM processing
- variant calling
- variant filtering
- functional annotation

The pipeline reflects real-world workflows used in human genetics and clinical genomics.

## 🚀 Key Features
- Process raw exome sequencing reads
- Align reads to a reference genome
- Perform post-alignment processing
- Call SNPs and indels
- Filter high-confidence variants
- Annotate coding variants for biological interpretation
